# Aula 16 - 06/11/2024

1. Para utilizar comentários no blade, utilizamos {{-- --}}
``` 
{{-- Esse é um comentário --}}
```

2. Para utilizar código php, colocamos o @php 
```
@php
    echo 'Texto de teste';
@endphp
```
3. Para exibir o código com formato php na tela 
@dd($fornecedores)

4. Para utilizar condicionais
@if
@elseif
@else
@endif
